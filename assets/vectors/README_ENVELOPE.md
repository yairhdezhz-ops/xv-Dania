# Icono de Sobre para Sección de Regalos

Coloca aquí el icono de sobre para la sección de Gifts & RSVP.

## Archivo necesario:

1. **envelope-icon.svg** (o .png de alta resolución)
   - Icono de sobre cerrado
   - Color: beige claro o off-white con sombras sutiles
   - Debe ser transparente
   - Tamaño recomendado: mínimo 60x60px
   - Estilo: simple, elegante, con apariencia ligeramente tridimensional

## Notas:
- Preferible formato SVG para mejor calidad y escalado
- Si usas PNG, debe ser de alta resolución (mínimo 2x el tamaño de visualización)
- El icono debe tener fondo transparente
- El color debe ser beige claro/off-white con sombras sutiles para dar profundidad
