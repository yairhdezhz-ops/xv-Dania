# Vectores para Sección de Contenido

Coloca aquí los siguientes archivos vectoriales:

## Archivos necesarios:

1. **content-frame.svg** (o .png de alta resolución)
   - Marco dorado para enmarcar el contenido de la sección de padres y padrinos
   - Color: dorado (#D4AF37 o similar)
   - Debe ser transparente
   - Tamaño recomendado: mínimo 550x400px
   - Forma: marco irregular/geométrico con esquinas redondeadas
   - Debe tener espacio interno suficiente para el texto

## Notas:
- Preferible formato SVG para mejor calidad y escalado
- Si usas PNG, debe ser de alta resolución (mínimo 2x el tamaño de visualización)
- El vector debe tener fondo transparente
- Este marco es diferente a `gold-frame.svg` (que se usa en el header)
