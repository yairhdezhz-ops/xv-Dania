# Vectores del Header

Coloca aquí los siguientes archivos vectoriales:

## Archivos necesarios:

1. **gold-frame.svg** (o .png de alta resolución)
   - Marco dorado geométrico irregular (hexagonal)
   - Color: dorado (#D4AF37 o similar)
   - Debe ser transparente
   - Tamaño recomendado: mínimo 400x300px

## Notas:
- Preferible formato SVG para mejor calidad y escalado
- Si usas PNG, debe ser de alta resolución (mínimo 2x el tamaño de visualización)
- El vector debe tener fondo transparente
