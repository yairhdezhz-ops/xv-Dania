# Imágenes del Header

Coloca aquí las siguientes imágenes:

## Archivos necesarios:

1. **floral-bouquet.png** (o .jpg, .webp)
   - Arreglo floral en estilo acuarela
   - Colores: púrpuras, rosas, cremas y verdes pastel
   - Debe ser transparente (PNG con fondo transparente recomendado)
   - Tamaño recomendado: mínimo 800x600px

## Notas:
- Acepta formatos: PNG, JPG, WEBP
- Si la imagen tiene fondo, debe ser transparente o blanco
- La imagen se escalará automáticamente según el diseño responsive
