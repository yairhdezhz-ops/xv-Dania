# Imágenes para Sección de Contenido

Coloca aquí las siguientes imágenes:

## Archivos necesarios:

1. **floral-decoration-2.png** (o .jpg, .webp)
   - Arreglo floral decorativo para la parte superior de la sección de padres y padrinos
   - Estilo acuarela con colores pastel (lavanda, púrpura, rosa, verde)
   - Debe ser transparente (PNG con fondo transparente recomendado)
   - Tamaño recomendado: mínimo 600x200px
   - Se posiciona en la parte superior, centrado

## Notas:
- Acepta formatos: PNG, JPG, WEBP
- Si la imagen tiene fondo, debe ser transparente o blanco
- La imagen se escalará automáticamente según el diseño responsive
- Esta imagen es diferente a `floral-bouquet.png` (que se usa en el header)
